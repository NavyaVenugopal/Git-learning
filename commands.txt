GIT 

To see git name and email

git config --list

Git account :

git config --global user.name "your_name"

git config --global user.email "your_email"

 
git commands

git init : To initialize the  git repository

git status : list all untracked files

git add -A : Adding all untracked files to staging area

git commit -m "commit message" : staging area to localrepository 

git remote add origin remote_path : For adding remote repository (git hub created repository)

git push origin master : for pushing local repository to remote repository (eg;vscode to github)
